# 系统要求

ezBookkeeping 支持在主流的操作系统上部署，同时支持在现代浏览器上访问其界面。以下列出了 ezBookkeeping 各个版本支持的操作系统和浏览器的最低版本要求。

## 支持部署 ezBookkeeping 服务端程序的操作系统

运行 ezBookkeeping 服务端的操作系统与 Golang 支持的操作系统相同，下表列出了 ezBookkeeping 各个版本依赖的 Golang 的版本以及支持操作系统的最低版本。

| ezBookkeeping 版本 | Golang 版本 | Linux Kernel 最低版本 | Windows 最低版本 | macOS 最低版本 |
| --- | --- | --- | --- | --- |
| v1.1.x 及更新 | 1.25 | ≥ 3.2 | ≥ Windows 10 / Windows Server 2016 | ≥ macOS 11 Big Sur |
| v0.8.x ~ v1.0.x | 1.24 | ≥ 3.2 | ≥ Windows 10 / Windows Server 2016 | ≥ macOS 11 Big Sur |
| v0.7.x | 1.23 | ≥ 2.6.32 | ≥ Windows 10 / Windows Server 2016 | ≥ macOS 11 Big Sur |
| v0.6.x | 1.22 | ≥ 2.6.32 | ≥ Windows 10 / Windows Server 2016 | ≥ macOS 10.15 Catalina |
| v0.5.x | 1.21 | ≥ 2.6.32 | ≥ Windows 10 / Windows Server 2016 | ≥ macOS 10.15 Catalina |
| v0.2.x ~ v0.4.x | 1.20 | ≥ 2.6.32 | ≥ Windows 7 / Windows Server 2008 R2 | ≥ macOS 10.13 High Sierra |
| v0.1.x | 1.16 | ≥ 2.6.32 | ≥ Windows 7 / Windows Server 2008 R2 | ≥ macOS 10.12 Sierra |

上述 Linux 版本仅表示 amd64、arm64 和 armv6/v7 架构下的版本，其他架构下支持的版本请参考 Golang 官方文档。更多 Golang 支持的操作系统的信息，请访问 Golang 官方文档：[https://go.dev/wiki/MinimumRequirements](https://go.dev/wiki/MinimumRequirements)。

## 支持访问 ezBookkeeping 界面的浏览器

如果你已经部署了 ezBookkeeping 的服务端程序，你可以直接使用你的浏览器访问 ezBookkeeping，下表列出了 ezBookkeeping 各个版本支持的浏览器的最低版本。

| ezBookkeeping 版本 | Chrome 最低版本 | Edge 最低版本 | Firefox 最低版本 | Safari 最低版本 |
| --- | --- | --- | --- | --- |
| v1.1.x 及更新 | ≥ 91 (2021年5月) | ≥ 91 (2021年4月) | ≥ 91 (2021年8月) | ≥ 15.4 (2022年3月) |
| v0.3.x ~ v1.0.x | ≥ 87 (2020年12月) | ≥ 88 (2020年12月) | ≥ 78 (2020年6月) | ≥ 14.0 (2020年9月) |
| v0.1.x ~ v0.2.x | ≥ 63 (2017年12月) | ≥ 18 (2018年10月) | ≥ 58 (2018年1月) | ≥ 11.3 (2018年3月) |
